Tortola

British Virgin Islands

Local tradition recounts that Christopher Columbus named the island Tortola, meaning "turtle dove" in Spanish. In fact, Columbus named the island as Santa Ana. Dutch colonists called it Ter Tholen, after a coastal island that is part of the Netherlands. When the British took over, the name evolved to Tortola.
